# AI-Powered Fraud Detection System

## Overview

This project focuses on detecting fraudulent transactions using machine learning techniques.

The dataset is highly imbalanced, containing only 81 fraudulent transactions out of 9,082 total records.

## Features

- Exploratory Data Analysis (EDA)
- Missing Value Handling
- Feature Engineering
- Correlation Analysis
- Feature Importance Analysis
- Logistic Regression Baseline
- XGBoost Classifier
- Scikit-Learn Pipelines

## Dataset Statistics

- Total Records: 9,082
- Genuine Transactions: 9,001
- Fraud Transactions: 81
- Fraud Rate: 0.89%

## Models Used

### Logistic Regression
Baseline model used for benchmarking.

### XGBoost
Advanced gradient boosting model used for final predictions.
